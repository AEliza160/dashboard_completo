# dashboard_completo
Este proyecto es un dashboard visual sobre un dataset de móviles.
El dataset contiene información sobre diferentes características de los móviles, como rango de precio, batería, RAM, memoria interna y otros.

El objetivo del proyecto fue analizar estos datos y mostrar información importante de forma visual para entender mejor los móviles según sus características.

Se crearon seis gráficos diferentes en un solo dashboard:

El primer gráfico muestra la cantidad de móviles en cada rango de precio.

El segundo gráfico muestra cómo se distribuyen las baterías de los móviles.

El tercer gráfico muestra la distribución de RAM entre los móviles.

El cuarto gráfico muestra los 10 valores más comunes de memoria interna.

El quinto gráfico muestra la batería promedio según el rango de precio.

El sexto gráfico muestra la RAM promedio por rango de precio en un gráfico de línea.


Todos los gráficos fueron personalizados con colores y etiquetas para que sean fáciles de leer.
El dashboard final se guardó como una imagen en alta calidad para poder compartirla o usarla en reportes.

Este proyecto sirve para ver rápidamente patrones en los datos de móviles, como qué rangos de precio son más comunes, qué baterías se usan más o cómo cambia la RAM según el precio.
